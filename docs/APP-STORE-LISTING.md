# App Store Connect — liste metinleri (iOS 1.0)

Bu dosyadaki metinleri [App Store Connect](https://appstoreconnect.apple.com) → **Omnifolio** → **Distribution / App Store** → **iOS 1.0 (Prepare for Submission)** sayfasındaki ilgili alanlara kopyalayın.

## Yapıştırma kontrol listesi

- [ ] Promotional Text (Türkçe veya İngilizce — primary dilinize göre)
- [ ] Description
- [ ] Keywords
- [ ] Support URL
- [ ] Marketing URL (isteğe bağlı — boş bırakılabilir)
- [ ] Copyright

Primary Language **Turkish** ise aşağıdaki **Türkçe** blokları kullanın. **English** ise **İngilizce** blokları kullanın; diğer dili isterseniz sonra **App Store** → **Localization** üzerinden ekleyebilirsiniz.
